What is Pixel Art?

Pixel art is a form of digital art where images are created and edited at the pixel level. Each individual pixel is deliberately placed to create forms, characters, and environments with a distinctive aesthetic.

Key Principles

  • Grid-Based Design: Every pixel is placed on a precise grid
  • Deliberate Placement: Each pixel serves a specific purpose
  • Limited Palettes: Constrained colour choices create harmony
  • Cultural Sensitivity: Inclusive design from the start
